hexo+github博客的美化及更换博客的主题

1.在hexo主题页挑选喜欢的主题

https://hexo.io/themes/

2.挑选到喜欢的主题后，点击主题下面的名称进入github仓库

3.点击code复制仓库地址https://github.com/Fechin/hexo-theme-diaspora.git

4.打开Hexo文件夹下的themes目录(f:\Blog\themes)，右键Git Bash Here,输入下载命令

git clone https://github.com/Fechin/hexo-theme-diaspora.git           #（刚才复制的仓库地址）

5.打开Hexo文件夹下的网站配置文件_config.yml

注意：blog目录中的config.yml为网站配置文件，themes里面的config.yml为主题配置文件

修改参数为

theme：hexo-theme-diaspora

6.在blog目录中，右键Git Bash Here输入

hexo g

hexo s

打开浏览器，输入http://localhost:4000/或者http://ip:4000/查看效果

7.满意的话部署到服务器

打开文件夹blog，右键Git Bash Here，输入

hexo clean（必须要做这一步，否则有时因缓存问题，服务器更新不了主题）

hexo g 

hero d

8.修改主题

在themes中主题配置文件config.yml中按照需求修改，注意用txt打开可能出错，下载一个vs code
