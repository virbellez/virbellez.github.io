三月 11, 2022

在windows部署Hexo+GitHubPages搭建个人博客网站

系统windows10 64专业版

参考链接https://zhuanlan.zhihu.com/p/26625249

1.在git和node.js官网下载安装包

https://git-scm.com/

https://nodejs.org/zh-cn/

2.击安装。一路next，可以自己选择安装路径

3.版本检查

node -v


git --version

4.Git与GitHub帐号绑定

到git安装文件夹，用鼠标右击以管理员方式打开Git Bash.exe

5.设置user.name和user.email配置信息：

git config --global user.name "GitHub用户名"

git config --global user.email "GitHub注册邮箱"

6.生成ssh密钥文件：

ssh-keygen -t rsa -C "GitHub注册邮箱"

直接三个回车

7.找到默认文件夹中生成的.ssh的文件夹中的id_rsa.pub密钥打开，将内容全部复制

8.打开github-setting-ssh and gpg keys，添加sshkeys

9.在Git Bash中检测GitHub公钥设置是否成功，输入 ssh git@github.com

ssh git@github.com

10.Node.js会包含环境变量及npm的安装，检测检测npm是否安装成功，在命令行中输入版本检查命令

npm -v

到这了，安装Hexo的环境已经全部搭建完成

11.在电脑里创建一个文件夹，命名为blog 我新建文件夹，我在F新建了blog文件夹

12.在blog文件夹中使用npm命令安装Hexo，注意这里要在blog文件夹中，按住shift，鼠标右键，选择在此处打开powershell

npm install -g hexo-cli

13.在blog文件夹中初始化博客

hexo init

14.检测我的网站雏形，分别按顺序输入以下三条命令

hexo new hello fantasy

hexo g

hexo s

15.在本地部署hexo完毕后，在本机用http://localhost:4000/可以正常打开网页

16.在blog根目录里的_config.yml文件称为站点配置文件

更改config站点配置文件

打开站点的配置文件_config.yml，翻到最后修改为

deploy:

type: git

repo: https://github.com/virbellez/virbellez.github.io.git #（这里填入你之前在GitHub上创建仓库的完整路径，记得加上 .git）（或者git@github.com:virbellez/virbellez.github.io.git）

branch: master

如果18步骤出现错误err: Error: Spawn failed
则是站点配置文件repo后面的仓库http模式更换为SSH模式就好了，改为git@github.com:virbellez/virbellez.github.io.git

17.安装git部署插件要安装在blog文件夹中

npm install hexo-deployer-git --save

安错文件夹后会报错：Deployer not found: git

18.安装完后在blog文件夹中输入三条命令

hexo clean

hexo g

hexo d

安装完后跳出弹窗

打开弹窗里面的链接输入弹窗中密码

点右边授权按钮

其实第三条的 hexo d 就是部署网站命令，d是deploy的缩写。完成后，打开浏览器，在地址栏输入你的放置个人网站的仓库路径，即 http://xxxx.github.io， 比如我的xxxx就是我的GitHub用户名：virbellez

备注：

如果是已有博客迁移到新电脑，需要在github-setting-ssh keys那里删掉旧的密码，将git生成的新密码填进去

将之前githun上面的文件clon下来，存在旁边，等hexo上线后，将旧的文件替换进，.deploy_git文件夹中

19.博客写文章

在blog目录输入命令hexo n “文章标题”会在项目\Hexo\source_post中生成“文章标题.md”文件，用编辑器打开编写即可

hexo n "文章标题"

书写好博客后，可以先上传到本地看看效果

hexo g

hexo s

上传到github

hexo clean

hexo g

hexo d
